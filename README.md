
## 🛠️ Tecnologías Utilizadas

- **Frontend**: React 18 + TypeScript
- **Routing**: React Router 6
- **Styling**: TailwindCSS + CSS Variables
- **Build Tool**: Vite
- **Icons**: Lucide React
- **Deployment**: Netlify


## 🚀 Comandos de Desarrollo

```bash
# Instalar dependencias
npm install

# Desarrollo local
npm run dev

# Build para producción
npm run build

# Preview del build
npm run preview

# Linting
npm run lint
```
